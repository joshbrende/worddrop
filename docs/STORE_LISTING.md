## Controls
### Desktop
*   **Arrow Left / Right (or A / D)**: Move Letter
*   **Arrow Down (or S)**: Fast Drop
*   **Spacebar**: Pause Game (or use on-screen button)

### Mobile
*   **Left / Right Buttons**: Move Letter
*   **Drop Button**: Hold to fast drop
*   **Tap Board**: Pause
